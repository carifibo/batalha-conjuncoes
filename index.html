Configurar Firebase

Faça isso uma vez. As mesmas credenciais vão em todos os dispositivos.

📋 Como obter as credenciais — 3 passos
1
Acesse console.firebase.google.com → "Criar um projeto". Dê qualquer nome e desative o Google Analytics.
2
No menu lateral: Criação → Realtime Database → Criar banco de dados. Escolha qualquer região e selecione "Modo de teste".
3
Volte ao menu lateral → engrenagem → Configurações do projeto. Desça até "Seus aplicativos" → clique em </> Web → registre o app → copie os valores do firebaseConfig abaixo.

⚠ Preencha todos os campos.

As credenciais são salvas no localStorage deste navegador.
